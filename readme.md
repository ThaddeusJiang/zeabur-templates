# zeabur templates

one-click deploy multiple services to the cloud

## Cobalt

- zeabur template: https://zeabur.com/templates/MMS61P
- cobalt github: https://github.com/imputnet/cobalt/

```sh
npx zeabur template update -c MMS61P -f cobalt.yml
```
